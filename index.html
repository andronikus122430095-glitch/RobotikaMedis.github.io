Smart Nurse Call System

Sistem pemanggilan perawat nirkabel berbasis Micro-ROS sebagai implementasi Robotika Medis pada Smart Hospital

Pendahuluan Deskripsi Tujuan dan Manfaat Arsitektur Sistem Perangkat Keras Perangkat Lunak Kesimpulan Dokumentasi

Pendahuluan

Dalam lingkungan rumah sakit, komunikasi antara pasien dan tenaga medis merupakan aspek krusial. Sistem pemanggil perawat konvensional memiliki keterbatasan, seperti minimnya informasi kondisi pasien dan kurangnya integrasi dengan sistem digital.

Oleh karena itu, proyek Smart Nurse Call System dikembangkan sebagai solusi berbasis robotika medis dan IoT untuk meningkatkan respons tenaga medis, efisiensi pelayanan, serta keselamatan pasien.

Deskripsi Proyek

  • ESP32 sebagai node klien
  • ROS 2 sebagai pusat kendali
  • Push button sebagai input pasien
  • LED dan buzzer sebagai notifikasi

Tujuan dan Manfaat

  • Mengembangkan sistem pemanggil perawat berbasis IoT dan embedded system
  • Mengintegrasikan mikrokontroler dengan antarmuka pengguna
  • Meningkatkan kecepatan dan akurasi respons tenaga medis
  • Menerapkan konsep robotika medis dalam sistem pelayanan kesehatan
  • Menjadi media pembelajaran implementatif pada mata kuliah Robotika Medis

Arsitektur Sistem

Sistem Smart Nurse Call terdiri dari beberapa komponen utama, yaitu mikrokontroler ESP32, tombol input pasien, buzzer dan LED indikator, serta tampilan LCD. Data dari pasien diproses oleh ESP32 dan dikirimkan melalui jaringan IoT untuk ditampilkan pada sistem monitoring.

Middleware robotik seperti ROS 2 dan micro-ROS memungkinkan sistem ini dikembangkan lebih lanjut menjadi bagian dari ekosistem robotika medis yang terdistribusi.

Perangkat Keras

  • ESP32 sebagai pengendali utama sistem
  • Push button sebagai input panggilan pasien
  • Buzzer dan LED sebagai indikator visual dan audio
  • Breadboard dan kabel jumper untuk perakitan rangkaian

Perangkat Lunak dan Teknologi

  • Arduino IDE untuk pemrograman ESP32
  • JavaScript, HTML, dan CSS untuk antarmuka web
  • GitHub Pages sebagai media publikasi website
  • Konsep IoT untuk komunikasi data
  • ROS 2 dan micro-ROS sebagai pengembangan lanjutan

Kesimpulan

Proyek Smart Nurse Call System menunjukkan bahwa penerapan robotika medis berbasis IoT mampu meningkatkan efisiensi dan kualitas pelayanan kesehatan. Sistem ini dapat dikembangkan lebih lanjut dengan integrasi sensor biometrik, kecerdasan buatan, dan sistem rumah sakit terpusat.
