Latar Belakang

Robot Operating System (ROS) adalah framework yang banyak digunakan dalam pengembangan sistem robotika, termasuk robotika medis. Salah satu konsep fundamental dalam ROS adalah pola komunikasi Publisher-Subscriber yang memungkinkan pertukaran data antar node secara asinkron.

Proyek ini mendemonstrasikan implementasi dasar dari konsep tersebut melalui sistem kontrol LED yang dikendalikan oleh push button, yang dapat menjadi fondasi untuk aplikasi robotika medis yang lebih kompleks.

Arsitektur Sistem

PUBLISHER NODE

Button Input

  • Membaca status push button
  • Mempublikasikan data ke topic
  • Debouncing untuk stabilitas
ROS Topic: /button_state

SUBSCRIBER NODE

LED Control

  • Subscribe ke topic button
  • Memproses data yang diterima
  • Mengontrol status LED

Komponen Utama

ROS Master: Koordinator komunikasi antar node
Publisher Node: Node yang membaca input dari push button dan mempublikasikan state
Subscriber Node: Node yang menerima data dan mengontrol LED
Topic: Channel komunikasi untuk pertukaran message

Implementasi

1. Publisher Node (button_publisher.py)

#!/usr/bin/env python
import rospy
from std_msgs.msg import Bool
import RPi.GPIO as GPIO

class ButtonPublisher:
    def __init__(self):
        rospy.init_node('button_publisher')
        self.pub = rospy.Publisher('/button_state', Bool, queue_size=10)
        self.button_pin = 17
        
        GPIO.setmode(GPIO.BCM)
        GPIO.setup(self.button_pin, GPIO.IN, pull_up_down=GPIO.PUD_UP)
        
    def publish_button_state(self):
        rate = rospy.Rate(10)  # 10 Hz
        while not rospy.is_shutdown():
            button_state = GPIO.input(self.button_pin) == GPIO.LOW
            self.pub.publish(button_state)
            rate.sleep()

if __name__ == '__main__':
    try:
        publisher = ButtonPublisher()
        publisher.publish_button_state()
    except rospy.ROSInterruptException:
        pass

2. Subscriber Node (led_subscriber.py)

#!/usr/bin/env python
import rospy
from std_msgs.msg import Bool
import RPi.GPIO as GPIO

class LEDSubscriber:
    def __init__(self):
        rospy.init_node('led_subscriber')
        self.led_pin = 27
        
        GPIO.setmode(GPIO.BCM)
        GPIO.setup(self.led_pin, GPIO.OUT)
        GPIO.output(self.led_pin, GPIO.LOW)
        
        rospy.Subscriber('/button_state', Bool, self.callback)
        
    def callback(self, msg):
        if msg.data:
            GPIO.output(self.led_pin, GPIO.HIGH)
            rospy.loginfo("LED ON")
        else:
            GPIO.output(self.led_pin, GPIO.LOW)
            rospy.loginfo("LED OFF")
            
    def spin(self):
        rospy.spin()

if __name__ == '__main__':
    try:
        subscriber = LEDSubscriber()
        subscriber.spin()
    except rospy.ROSInterruptException:
        pass
