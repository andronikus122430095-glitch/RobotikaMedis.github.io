Smart Nurse Call System

Sistem pemanggilan perawat nirkabel berbasis Micro-ROS sebagai implementasi Robotika Medis pada Smart Hospital

Pendahuluan Deskripsi Tujuan Dokumentasi

Pendahuluan

Sistem Nurse Call konvensional memiliki keterbatasan dalam fleksibilitas dan integrasi data. Proyek ini mengembangkan sistem nirkabel berbasis Micro-ROS dan ROS 2 untuk mendukung Smart Hospital.

Deskripsi Proyek

  • ESP32 sebagai node klien
  • ROS 2 sebagai pusat kendali
  • Push button sebagai input pasien
  • LED dan buzzer sebagai notifikasi

Tujuan Pengembangan

  • Mengimplementasikan Micro-ROS
  • Menerapkan sistem kendali tertutup
  • Mendukung pengembangan Robotika Medis
